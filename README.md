# Health Tracker

## Overview
This is a simple Python-based Health Tracker application built with Tkinter.  
It allows users to monitor daily health metrics such as steps, water intake, sleep hours, and exercise.  
Data is stored in a CSV file for easy tracking of past records.

## Features
- Track daily steps, water intake, sleep, and exercise
- Persistent storage using CSV
- Simple, intuitive GUI using Tkinter
- Easy to run and use

## Tech Used
- Python
- Tkinter
